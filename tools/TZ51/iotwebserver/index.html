L'internet des objets pour tous

L'ESP8266

Ce module conçu pour être interfacé à une carte Arduino UNO ou MEGA ( via un shield EASYBOT1 ou EASYCONx)   contient un serveur HTTP qui vous délivre des pages HTML comme celle que vous êtes en train de lire. De plus il intègre un serveur WebSocket qui peut établir une connexion persistante avec votre navigateur. Ceci permet une intéraction entre vos montages physiques et n'importe quel navigateur supportant la technologie des WebSockets.
